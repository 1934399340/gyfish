## 📋 使用说明

### 访问后台管理系统
1. 打开：`https://gyfish-4gbxsy3dc619f1e8-1416769265.tcloudbaseapp.com/admin/unified-login.html`
2. 选择登录方式：
   - **邮箱登录**：使用管理员邮箱和密码
   - **GitHub 登录**：使用 GitHub 账号一键登录
3. 登录成功后进入管理后台仪表盘

### 测试系统功能
1. 打开：`https://gyfish-4gbxsy3dc619f1e8-1416769265.tcloudbaseapp.com/admin/system-test.html`
2. 运行各项测试检查系统状态
3. 验证联系表单和其他功能

### 配置邮件服务
由于安全原因，邮件发送功能需要配置 QQ 邮箱的 SMTP 授权码：

1. 登录 QQ 邮箱
2. 进入设置 → 账户
3. 开启 "POP3/SMTP服务"
4. 获取授权码
5. 在 `functions/send-email/index.js` 中替换 `your_qq_auth_code`
